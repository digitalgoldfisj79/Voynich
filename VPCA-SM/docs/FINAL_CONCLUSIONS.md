## RECOMMENDATIONS FOR FUTURE RESEARCHERS

### If Using Our Methods:

1. **Follow two-phase design**
   - Phase 1: Exploratory (hypothesis generation)
   - Phase 2: Confirmatory (validation)
   - Never conflate the two

2. **Apply QC checklist preemptively**
   - Before claiming findings
   - Check: data mining, independence, comparisons
   - Start conservative

3. **Pre-register confirmatory tests**
   - Lock predictions before testing
   - Timestamp on GitHub/OSF
   - Follow original predictions

4. **Report all results**
   - Including null findings
   - No selective reporting
   - Null results are valuable

5. **Appropriate confidence levels**
   - Exploratory: max 20-30% semantic
   - Single validation: max 50-60%

   - Be conservative
