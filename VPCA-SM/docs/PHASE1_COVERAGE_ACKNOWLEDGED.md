## THE CRITICAL QUESTION

**Does testing 'e' position on VPCA-analyzed data count as "independent validation"?**

### Argument FOR Legitimacy:

**Phase 1 examined different patterns:**
- VPCA analyzed morphological structure (C/P/V/A)
- VPCA did NOT examine 'e' position patterns specifically
- 'e' was not identified as a special pattern in Phase 1
- Testing 'e' position is a different, independent hypothesis

**Analogy:**
- If you analyze a text for word frequency (Phase 1)
- Then test grammatical patterns (Phase 2)
- The second test is legitimate because it's a different hypothesis

**Precedent:**
- Common in science to test multiple hypotheses on same dataset
- Legitimate if hypotheses are independent and pre-registered
- We pre-registered 'e' predictions before testing

### Argument AGAINST Legitimacy:

**Same data examined:**
- Phase 1 looked at these tokens for morphology
- Phase 2 looked at same tokens for position
- Not truly "independent" data

**Risk of overfitting:**
- Having seen the data for any purpose may influence pattern-finding
- Even subconsciously
- "Hold-out" traditionally means completely unseen

**Conservative standard:**
- Safest approach: only count truly unseen data
- External corpus is only truly independent validation
- All manuscript sections are "contaminated" to some degree
