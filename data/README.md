# Data Files

This directory contains corpus files and input data for the analyses.

## Files Needed

Add these files from the main branch:
- `voynich_tokens.txt` - Voynich manuscript token list (copy from `p6_voynich_tokens.txt`)
- `latin_abbrev_expanded.txt` - Expanded Latin corpus
- `occitan_medieval_stems.txt` - Medieval Occitan stems

## Usage

These files are used by:
- Morphology analysis scripts
- Compression testing scripts
- Master analysis script (`do_real_work.py`)
